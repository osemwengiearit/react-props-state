# React Props & State App

This project is a React application built to demonstrate the use of props, state, and conditional rendering. It was developed as part of the Axia Africa React Module.

## Concepts Covered

- Passing data with props
- Managing component state with useState
- Conditional rendering
- Handling user interactions
- Component composition

## Tech Stack

- React
- Vite
- JavaScript (ES6)

## Setup Instructions

```bash
npm install
npm run dev
```
